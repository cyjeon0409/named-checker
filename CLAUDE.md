# 네임드 기업 판정기

KDT 고성과 과정 참여기업의 네임드 여부를 자동 판정하는 웹앱입니다.

## 실행 방법

사용자가 "실행해줘", "띄워줘", "시작해줘" 등을 요청하면 아래 순서대로 실행하세요.

### 1단계: 패키지 설치
```
pip install -r requirements.txt
```

### 2단계: 서버 실행
```
python main.py
```

실행 후 브라우저에서 http://localhost:8080 으로 자동으로 열립니다.

## 주요 기능

- **기업 판정 탭**: 기업명 입력 → 임직원/투자금/매출/대기업집단 기준으로 네임드 여부 자동 판정
- **채용 공고 탭**: 기업명 또는 직무 키워드로 사람인·잡코리아·원티드 채용 공고 검색

## 판정 기준 (아래 중 1개 이상 충족 시 네임드)

1. 임직원 100명 이상
2. 누적 투자금 50억 이상 (시리즈 A~E)
3. 연매출 100억 이상
4. 공정위 지정 대기업집단 계열사
